# REST Meetups API

## Features:
- Express.js + TypeScript
- Prisma + PostgreSQL
- Architecture: route-controller-service-repository
- DTO validation + middleware
- Swagger docs
- JWT Auth + Passport.js
- Pagination, search, sorting via query params
